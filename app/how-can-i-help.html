Helping out and getting involved

Join the hordes

Come and join the horde in the Hub's War Room. Pins on the globe represent peers: green online, red offline. We aim to fill the globe up with lots of green pins with no bald spots. We collaborate in the built in chat or on our Slack Channel to find new folks to create peers in the empty patches.

Developers

Next we need developers! We need Ethereum blockchain and Solidity savvy engineers, P2P researchers to help us experiment and optimize DHT and Gossip protocols. We need C/C++, Java, Javascript, Golang programmers. If you want to get involved as a developer first give Subutai a try as a user. Then take a look at our list of GitHub projects below. Come onto our Slack channel and let us know if you need help diving in. We’re here and waiting for you!

Register on the Subutai Hub this month using the code SITEOCT17, and we’ll give you some goodwill to trade cloud resources with others on the system. Happy sharing!

Kick the tires and give us feedback


We need feedback from people who create peers, share resources with friends, and create cloud environments. Tell us what you think by hitting the “Rate us” widget in the War Room.

If you get stuck and need help, just ping us in the “Help Desk” room or feel free to get on Slack.

For sysadmins and techies, peer installation is straightforward with many options. We’re working on finishing touches with different platforms and package managers so if anyone wants to help out we appreciate it.


For those of you who want to help, but are not technically savvy, don’t worry you can still help. We need to make sure anyone can provide and utilize resources. We’re working on an easy to use peer launcher, however it’s a bit raw and in beta. It should help you setup a peer inside a virtual machine to sell resources to others and earn Goodwill contributing idle resources to open source projects. There may be problems with the launcher since everyone’s setup is different and we have not worked out the bugs. However if you could help us out by testing them that would be awesome.


We need GitHub projects to register for the OS contribution program and provide blueprints for their applications so they can be made available to all users in the Subutai Bazaar. The Subutai FOSS Contribution Program explains this in detail.

Anyone can then fire up open source cloud applications while sharing resources.
