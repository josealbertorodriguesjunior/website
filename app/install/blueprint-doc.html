Here​ ​ is​ ​ simple​ ​ Subutai​ ​ blueprint​ ​ which​ ​ creates​ ​ 1 ​ ​ container​ ​ inside​ ​ one​ ​ peer.

                                {
                                    "name": "SubutaiBlueprint",
                                    "description": "Environment configuration test blueprint​",
                                    "nodes": [
                                        {
                                            "name": "backup-server",
                                            "template": "ubuntu16",
                                            "peer_group": "HTTP-GROUP",
                                            "size": "MEDIUM"
                                        }
                                    ],
                                    "peerGroups": [
                                        {
                                            "name": "HTTP-GROUP",
                                            "pricingPref": {
                                                "maxPrice": 50
                                            },
                                            "hwPreferences": {
                                                "avgCpuload": "50",
                                                "minFreeRam": "128MiB",
                                                "minFreeDiscSpace": "500MiB"
                                            }
                                        }
                                    ]
                                }
Parameter Value Description
name String Environment​ ​name
Description Text Environment​ ​description​ ​can​ ​be​ ​any text.
nodes Json​ ​ (container​ ​ parameters) name​​ ​- ​value​ ​can​ ​be​ ​any​ ​string
template​​ ​- ​template​ ​ name​ ​ from​ ​ which template​ ​ will​ ​be​ ​clon​ ​container.
peer_group​​ ​ - ​to​ ​which​ ​peer​ ​group container​ ​ belong.​ ​ String​ ​ value.
size​​ ​- ​container​ ​size,​ ​values:​ ​ TINY, MEDIUM, SMALL,​ ​ LARGE,​ ​ HUGE
peerGroups Json​ ​ array name​​ ​- ​any​ ​string pricingPref.maxPrice​ ​- ​if​ ​peer is​ ​paid​ ​ you​ ​ should​ ​put​ max price.
hwPreferences Json avgCpuLoad​ ​- ​CPU​ ​load​ ​avarage in​ ​percentage.
minFreeRam ​- ​minimum​ ​free​ ​ram in​ ​MB
minFreeDiskSpace​ ​- ​minimum​ ​free disk​ ​space​ ​in​ ​MB
