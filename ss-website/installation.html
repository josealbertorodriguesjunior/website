Installation

Subutai Social can directly install onto bare metal, but for convenience we've packaged everything into a virtual machine. We did this so you don't have to change anything and it's effortless to install and remove. We made some easy to use installers to take the hassle out of setting up everything for Windows and Mac OS X users. *Nix users please see the Readme.md

Ubuntu/Debian P2P:

To install latest p2p package you need to add PPA repository into your Ubuntu Linux software sources list by executing following commands:
sudo add-apt-repository ppa:subutai-social/subutai
sudo apt-get update

Now you can install stable p2p by running
sudo apt-get install subutai-p2p

If you wish to install development package with latest changes you can do the following:
sudo apt-get install subutai-p2p-dev

After installation you should be ready to share resources. The following components are installed onto your system if not already present:

  1. Google Chrome Browser
  2. VirtualBox Hypervisor
  3. Subutai Social E2E Browser Plugin for Chrome
  4. Subutai Social Resource host (RH) VM Image for Virtual Box
  5. Subutai Social Management host (MH) Template Image
  6. OS Specific Subutai Social P2P Executable
  7. Subutai Social Tray Application
